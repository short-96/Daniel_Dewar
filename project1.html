Program Assignment 1: Event-Driven Programming

Description of project:

Developed a Java application that creates a standalone GUI application that simulates an e-store. The application allows the user to place in-stock items into a shopping cart and once all items are included, total all costs (including tax), produce an invoice, and append a transaction log file.

Skills learned from completing project:

  • Java programming
  • GUI development
  • File processing
  • Simulating real-world applications
  • Documentation and systematic testing
